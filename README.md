# Commands:

- Base64 Encode
- Base64 Decode
- JWT Decode

# Keybindings:

### Mac

`cmd+e cmd+e` Encode

`cmd+e cmd+d` Decode

`cmd+e cmd+j` Encode JWT

### Win/Linux

`ctrl+e ctrl+e` Encode

`ctrl+e ctrl+d` Decode

`ctrl+e ctrl+d` Encode JWT
